# 🤖 AI Coding Assistant

An advanced AI-powered coding assistant built with Streamlit, Groq LLaMA 3, and Python to help developers and learners generate, explain, debug, and understand code faster with a modern futuristic UI.

---

## ✨ Features

* 💻 **AI Code Generation**
  Generate clean and production-ready code in multiple programming languages.

* 📖 **Code Explanation**
  Get detailed explanations for any code snippet.

* 🎯 **Beginner-Friendly Explanations**
  Understand complex code in simple terms.

* 🐛 **AI Debugging Assistant**
  Detect bugs and get corrected code instantly.

* 💡 **Programming Concept Tutor**
  Learn coding concepts with examples and easy explanations.

* 🎨 **Modern Glassmorphism UI**
  Beautiful futuristic interface with animated effects and custom backgrounds.

* ⚡ **Fast AI Responses**
  Powered by Groq API using LLaMA 3.3 70B model.

---

## 🛠️ Tech Stack

* 🐍 Python
* 🎈 Streamlit
* 🤖 Groq API (LLaMA 3)
* 🎨 HTML + CSS Custom Styling
* 🔐 dotenv for environment variables

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Soumyo2005/ai-coding-bot.git
cd ai-coding-bot
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

### 3️⃣ Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Mac/Linux

```bash
source venv/bin/activate
```

### 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Setup API Key

Create a `.env` file in the root directory and add:

```env
GROQ_API_KEY=your_api_key_here
```

⚠️ Never upload your `.env` file to GitHub.

---

## ▶️ Run the Application

```bash
streamlit run app.py
```
